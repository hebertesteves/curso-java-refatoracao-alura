# 📘 Curso Java — Refatoração e Boas Práticas

Repositório criado para armazenar os códigos, exemplos e práticas desenvolvidas durante o curso **Java e refatoração: melhorando códigos com boas práticas**, da **Alura**.

Este projeto tem como objetivo consolidar conceitos de **refatoração, boas práticas e organização de código em Java**, com foco em legibilidade, manutenção, testes automatizados e aplicação de padrões de projeto.

---

## 🧠 Conteúdos abordados

Durante o curso, foram estudados os seguintes tópicos:

- Refatoração de código Java  
- Aplicação de boas práticas de programação  
- Estruturação eficiente de métodos e classes  
- Princípios **SOLID**  
- Criação e modelagem de domínios  
- Aplicação do padrão de projeto **Command**  
- Uso do **Pattern Matching for Switch** do Java 15  
- Escrita de testes automatizados  
- Testes unitários com **JUnit e Mockito**  
- Organização e manutenção de código legível e escalável
